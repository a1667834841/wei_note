# 微信读书笔记助手

![微信读书笔记助手预览](https://img.ggball.top/picGo/预览.png)

一款支持markdown格式的微信读书笔记助手，支持左侧阅读，右侧记笔记，减少不同软件的切换时间，提高效率。

## 功能特点

- 支持左右分屏阅读和记笔记
- 支持markdown格式实时预览
- 支持导出markdown文件
- 支持快捷键开启/关闭笔记面板
